# Visualization-Tool-for-Electric-Vehicle-Charge-and-Range-Analysis

A Tableau-based analytics project focused on Electric Vehicle (EV) charging infrastructure, vehicle performance, battery efficiency, and market trends through interactive dashboards and visual storytelling.

## Project Overview

The rapid adoption of Electric Vehicles (EVs) has generated large volumes of data related to charging infrastructure, battery efficiency, vehicle range, and market trends. This project leverages Tableau to transform raw EV data into actionable insights through interactive dashboards and stories.

The objective is to help stakeholders understand charging infrastructure availability, vehicle performance, charging efficiency, and EV adoption patterns through data-driven visualizations.
